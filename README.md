# Movie Data Mashup & NLP Joke Generator

A Python project that combines movie data analysis with entertainment generation by fetching movie information and creating humorous mashups with jokes.

# Project Overview
This project demonstrates data science fundamentals by analyzing movie plot text, processing API data, and generating entertaining content. Given a movie title, the system:
1. Fetches movie metadata from the OMDb API (ratings, plot, year, etc.)
2. Analyzes plot text using natural language processing techniques
3. Mines keywords by ranking words by length and relevance
4. Searches for related content using the icanhazdadjoke API
5. Generates formatted output with highlighted keywords and contextual responses


# Technical Stack
- Language: Python 
- APIs: OMDb API, icanhazdadjoke.com API
- Libraries: requests, json, re, typing
- Data Processing: JSON parsing, text analysis, regular expressions
- Platform: Google Colab compatible


# Installation & Setup

# Prerequisites:
- Python 3.6+
- Internet connection for API calls
- OMDb API key 

# Quick Start:
1. Get your free OMDb API key from http://www.omdbapi.com/apikey.aspx
2. Copy the code to Google Colab or your Python environment
3. Run the code and start using!


# How to Run?
1. Open the `.ipynb` file in Google Colab or Jupyter Notebook.
2. Follow the steps in the notebook to run each cell.
3. Enter a movie title when prompted and enjoy the output!
